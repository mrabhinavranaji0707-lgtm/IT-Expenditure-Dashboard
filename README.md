# IT-Expenditure-Dashboard
## 📌 Overview
This project showcases an interactive Power BI dashboard designed to analyze IT expenditure across different business dimensions. It helps track and compare **Actual, Plan, and Forecast** values to identify trends, variances, and key cost drivers.

---

## 🎯 Objectives
- Monitor overall IT spending
- Identify variance between Actual vs Plan
- Track trends over time
- Enable detailed drilldown analysis for better decision-making

---

## 📂 Dashboard Structure

### 1️⃣ Executive Summary
- High-level KPIs (Actual, Plan, Forecast, Variance)
- Spend distribution by Business Area and Region

### 2️⃣ Variance Analysis
- Waterfall chart for variance breakdown
- Identification of top overspending areas
- Comparative matrix (Actual vs Plan)

### 3️⃣ Trend Analysis
- Monthly trend of Actual vs Plan vs Forecast
- Cumulative spending analysis
- Variance trends over time

### 4️⃣ Detailed Drilldown
- Hierarchical view (Date → Business Area → Region → Category)
- Deep dive into cost components
- Export-ready detailed table

---

## 🔑 Key Features
- Interactive slicers for dynamic filtering  
- Variance and percentage calculations

---

## 📈 Insights Delivered
- Identification of overspending areas  
- Regional and business-wise cost contribution  
- Trend-based analysis for better forecasting

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use slicers to explore different views  
4. Drill down into visuals for detailed insights
